# Responsive B2B E-Commerce & Supplier Marketplace Portal

A modern, highly responsive, and feature-rich B2B (Business-to-Business) E-commerce landing page built using **Tailwind CSS** and **Vanilla JavaScript**. This platform connects global buyers with international suppliers across multiple product categories.

Features

- **Responsive Top Navigation & Search Bar:** Features a categorized dropdown search layout alongside an intuitive, JavaScript-driven mobile burger menu toggle for seamless handheld viewing.
- **Dynamic Category & Hero Section:** Includes a sidebar navigation system mapping primary industry sectors linked alongside a trending banner highlight for consumer electronics.
- **Flash Deals with Countdown Timer:** Displays active supplier discounts with an integrated grid showcasing time-limited promotional offers.
- **Industry-Specific Showcase Grids:** Structured layout modules specifically segmenting product divisions such as *Home & Outdoor* and *Consumer Electronics & Gadgets* with base pricing markers.
- **Request for Quote (RFQ) Form Section:** A functional form container that enables business clients to send sourcing requests directly to global suppliers by specifying item names, details, quantities, and packaging units.
- **Recommended Items Grid:** A clean product matrix displaying varied trending inventory, from apparel and travel gear to household appliances.
- **Global Suppliers Registry & Footers:** A comprehensive regional directory highlighting international trade nodes by flag icons, backed by multi-column informative footer matrices and a newsletter subscription system.

Technologies Used

- **HTML5:** Structured semantic layouts.
- **Tailwind CSS v2.2:** Utility-first utility classes ensuring rapid interface layout and fluid responsive behavior.
- **Font Awesome v6.0 & Inline SVGs:** Vector iconography supporting interface headers, social indicators, and user dashboards.
- **JavaScript (Vanilla):** Light execution handling mobile layout menu expansion toggle interactions.
